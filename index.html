MIL記号 論理回路シミュレーター

INPUT / AND / OR / NOT / OUTPUT をキャンバス上へ追加し、MIL記号の形で回路を構築できるスタンドアロンWebアプリです。出力ピンから入力ピンへドラッグして配線し、INPUT をダブルクリックすると、信号の 0 / 1 がリアルタイムに下流へ伝播します。

単一HTML 外部ライブラリなし MIL記号 ドラッグ結線 リアルタイム信号可視化
コンポーネント追加
INPUT をダブルクリックで 0 / 1 切替、出力ピンから入力ピンへドラッグで結線

回路キャンバス

ドラッグ中の要素は最前面に表示されます。入力ピンは1本のみ接続可能です。

操作手順

  1. ツールバーから部品を追加します。
  2. ゲートをドラッグして好きな位置へ配置します。
  3. 右側の出力ピンから左側の入力ピンへドラッグして接続します。
  4. INPUT をダブルクリックして 0 / 1 を切り替えると、結果が即時に伝播します。

実装メモ

  • 未接続の入力は 0 として扱います。
  • 1つの入力ピンには1本だけ接続できます。新しい配線で上書きされます。
  • 循環回路による無限再計算を防ぐため、内部伝播回数に上限を設けています。
本アプリは要件定義書に基づき、HTML / CSS / JavaScript を1ファイルへ統合し、外部依存なしで動作するよう実装しています。
